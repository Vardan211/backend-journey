# Учебные Задания

Папка `assignments` повторяет учебные модули из `ROADMAP.md`.

Каждая папка модуля хранит маленькие домашние задания, которые брат делает через отдельную ветку и pull request.

```text
assignments/
  01-csharp-foundations/
  02-tools-and-debugging/
  03-http-and-api/
  04-sql-postgresql/
  05-ef-core-postgresql/
  06-tests-quality/
  07-docker-production/
```

Важно: не все модули заполняются заранее. В GitHub заранее держим только ближайшие 2-4 issue, а будущие задания добавляем после ревью предыдущих работ.

Папка `projects` живет отдельно, потому что большие проекты могут проходить через несколько модулей. Например, `todo-api` может начаться как API в памяти, потом получить PostgreSQL, EF Core, тесты и Docker.
