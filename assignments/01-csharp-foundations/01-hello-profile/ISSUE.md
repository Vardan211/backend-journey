# HW01: Hello Profile console app

## Goal

Build a console app that asks for name and age, validates the input, and prints a short profile message.

## Acceptance Criteria

- [ ] Runs with `dotnet run`
- [ ] Asks for the user's name
- [ ] Asks for the user's age
- [ ] Handles empty name
- [ ] Handles invalid age like `abc`
- [ ] Handles age below 0
- [ ] Handles age above 120
- [ ] Prints how old the user will be next year for valid input
- [ ] Has at least 3 small commits
- [ ] PR includes reflection answers

## Reflection Questions

1. Which part was easiest?
2. Which part was most confusing?
3. What error did you meet and how did you fix it?
4. Which test inputs did you try?
5. Which line of your code are you least sure about?
