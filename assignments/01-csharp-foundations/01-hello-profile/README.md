# HW01: Hello Profile

## Цель

Сделать первую консольную программу на C#, которая спрашивает имя и возраст, проверяет ввод и выводит короткое сообщение.

## Что Нужно Сделать

Программа должна:

1. спросить имя;
2. спросить возраст;
3. если имя пустое - вывести ошибку;
4. если возраст не число - вывести ошибку;
5. если возраст меньше 0 или больше 120 - вывести ошибку;
6. иначе вывести сообщение:

```text
Привет, Артем. Через год тебе будет 16.
```

## Как Создать Проект

Из корня репозитория:

```powershell
cd assignments/01-csharp-foundations/01-hello-profile
dotnet new console -n HelloProfile
cd HelloProfile
dotnet run
```

## Как Работать Через Git

Создать ветку:

```powershell
git checkout -b hw/01-hello-profile
```

Делать маленькие коммиты:

```powershell
git add .
git commit -m "Create console project"
git commit -m "Add name input"
git commit -m "Validate age input"
```

Отправить ветку:

```powershell
git push -u origin hw/01-hello-profile
```
