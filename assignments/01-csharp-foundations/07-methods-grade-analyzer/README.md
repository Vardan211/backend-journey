# HW07: Methods Grade Analyzer

## Цель

Сделать консольную программу для анализа оценок и вынести повторяющуюся логику в методы.

Это задание тренирует:

- методы;
- параметры;
- возвращаемые значения;
- `List<int>`;
- разбиение большой программы на маленькие части.

## Теория Перед Практикой

Перед кодом нужно понять:

1. Метод - это именованный кусок логики.
2. Параметры - это данные, которые метод получает.
3. `return` возвращает результат из метода.
4. Если весь код лежит в одном месте, его трудно читать и проверять.

Мини-пример для понимания, не готовое решение:

```csharp
static bool IsPassingGrade(int grade)
{
    return grade >= 3;
}
```

## Что Нужно Сделать

Программа должна:

1. попросить пользователя вводить оценки от `1` до `5`;
2. ввод пустой строки должен завершать ввод оценок;
3. если введено не число - вывести ошибку и спросить снова;
4. если оценка меньше `1` или больше `5` - вывести ошибку и спросить снова;
5. после ввода показать:
   - количество оценок;
   - среднюю оценку;
   - минимальную оценку;
   - максимальную оценку;
   - сколько оценок `5`;
6. если оценок нет - вывести понятное сообщение.

Нужно вынести минимум 4 метода:

- метод чтения одной оценки;
- метод расчета средней оценки;
- метод поиска минимальной оценки;
- метод поиска максимальной оценки.

## Как Создать Проект

Из корня репозитория:

```powershell
cd assignments/01-csharp-foundations/07-methods-grade-analyzer
dotnet new console --output .
dotnet run
```

## Важное Правило

Не писать всю программу в одном большом блоке. Главная цель задания - научиться выделять методы.
