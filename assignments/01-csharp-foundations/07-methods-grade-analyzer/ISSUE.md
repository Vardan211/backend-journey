# HW07: Methods Grade Analyzer

## Goal

Build a console grade analyzer and extract repeated logic into methods.

## Acceptance Criteria

- [ ] Runs with `dotnet run`
- [ ] Lets the user enter multiple grades
- [ ] Empty line finishes grade input
- [ ] Handles invalid input like `abc`
- [ ] Handles grade below `1`
- [ ] Handles grade above `5`
- [ ] Prints grade count
- [ ] Prints average grade
- [ ] Prints minimum and maximum grade
- [ ] Prints how many grades are `5`
- [ ] Has at least 4 actions extracted into methods
- [ ] Has at least 3 small commits
- [ ] PR includes reflection answers

## Reflection Questions

1. What is a method parameter?
2. What is a return value?
3. Which method in your code is clearest?
4. Which method in your code is least clear?
5. What became easier after splitting code into methods?
