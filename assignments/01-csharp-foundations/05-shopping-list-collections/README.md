# HW05: Shopping List Collections

## Цель

Сделать консольный список покупок, чтобы отдельно потренировать `List<string>`: добавление, просмотр, поиск, удаление и очистку.

Это задание тренирует:

- `List<string>`;
- `Add`;
- `Remove`;
- `Contains`;
- `foreach`;
- меню и проверки ввода.

## Теория Перед Практикой

Перед кодом нужно понять:

1. `List<string>` хранит много строк в одном месте.
2. `Add` добавляет новый элемент в список.
3. `Remove` удаляет элемент, если он найден.
4. `foreach` проходит по всем элементам списка.
5. Список в памяти пропадает после завершения программы.

Мини-пример для понимания, не готовое решение:

```csharp
var items = new List<string>();
items.Add("bread");

foreach (var item in items)
{
    Console.WriteLine(item);
}
```

## Что Нужно Сделать

Программа должна показывать меню:

```text
1. Добавить покупку
2. Показать список
3. Найти покупку
4. Удалить покупку
5. Очистить список
0. Выйти
```

Программа должна:

1. повторять меню, пока пользователь не выберет `0`;
2. добавлять покупку в `List<string>`;
3. не добавлять пустую покупку;
4. показывать все покупки;
5. если список пуст - выводить понятное сообщение;
6. искать покупку по названию;
7. удалять покупку по названию;
8. если покупка не найдена - выводить понятное сообщение;
9. очищать весь список после подтверждения;
10. неизвестную команду обрабатывать как ошибку.
