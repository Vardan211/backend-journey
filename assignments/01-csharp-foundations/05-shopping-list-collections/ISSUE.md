# HW05: Shopping List Collections

## Goal

Build a console shopping list to practice `List<string>` operations: add, list, search, remove, and clear.

## Acceptance Criteria

- [ ] Runs with `dotnet run`
- [ ] Stores items in `List<string>`
- [ ] Shows the menu until the user chooses `0`
- [ ] Adds an item
- [ ] Handles empty item
- [ ] Lists all items
- [ ] Shows a clear message when the list is empty
- [ ] Searches item by name
- [ ] Removes item by name
- [ ] Clears the list
- [ ] Handles unknown menu command
- [ ] Has at least 3 small commits
- [ ] PR includes reflection answers

## Reflection Questions

1. Why does this program need `List<string>`?
2. How is `Add` different from `Remove`?
3. What happens during `foreach`?
4. What happens to the list after the program exits?
5. Which list operation was most confusing?
