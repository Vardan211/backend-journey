# HW08: Contact Book With Classes

## Цель

Переделать идею списка контактов так, чтобы контакт был не одной строкой, а отдельным типом `Contact`.

Это задание тренирует:

- `class`;
- свойства;
- конструктор;
- `List<Contact>`;
- методы;
- простую проверку данных.

## Теория Перед Практикой

Перед кодом нужно понять:

1. Класс - это свой тип данных.
2. Объект - это конкретное значение этого типа.
3. Свойства хранят данные объекта.
4. Конструктор помогает создать объект сразу с нужными значениями.
5. `List<Contact>` хранит не строки, а контакты.

Мини-пример для понимания, не готовое решение:

```csharp
class Book
{
    public string Title { get; }

    public Book(string title)
    {
        Title = title;
    }
}
```

## Что Нужно Сделать

Программа должна показывать меню:

```text
1. Добавить контакт
2. Показать все контакты
3. Найти контакт по имени
4. Удалить контакт по имени
0. Выйти
```

Нужно создать класс `Contact` со свойствами:

- `Name`;
- `Phone`;
- `Email`.

Программа должна:

1. повторять меню, пока пользователь не выберет `0`;
2. добавлять контакт через создание объекта `Contact`;
3. не добавлять контакт с пустым именем;
4. не добавлять контакт с пустым телефоном;
5. email может быть пустым;
6. показывать все контакты в понятном виде;
7. искать контакт по имени;
8. удалять контакт по имени;
9. если контакт не найден - выводить понятное сообщение;
10. если выбрана неизвестная команда - выводить ошибку и снова показывать меню.
