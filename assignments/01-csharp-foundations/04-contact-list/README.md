# HW04: Contact List

## Цель

Сделать консольную программу "Список контактов", где пользователь может добавлять контакты, смотреть список и искать контакт по имени.

Это задание тренирует:

- `List<T>`;
- меню в консоли;
- цикл `while`;
- добавление элементов в список;
- поиск по строке.

## Теория Перед Практикой

Перед кодом нужно понять:

1. `List<string>` хранит несколько строк.
2. Меню - это цикл, который показывает варианты действий, пока пользователь не выберет выход.
3. Поиск можно сделать через перебор списка.
4. На этом задании контакт можно хранить одной строкой, например `Артем - +79990000000`.

Мини-пример для понимания, не готовое решение:

```csharp
var names = new List<string>();
names.Add("Артем");

foreach (var name in names)
{
    Console.WriteLine(name);
}
```

## Что Нужно Сделать

Программа должна показывать меню:

```text
1. Добавить контакт
2. Показать все контакты
3. Найти контакт по имени
0. Выйти
```

Программа должна:

1. повторять меню, пока пользователь не выберет `0`;
2. добавлять контакт с именем и телефоном;
3. не добавлять контакт с пустым именем;
4. не добавлять контакт с пустым телефоном;
5. показывать все контакты;
6. если контактов нет - писать, что список пуст;
7. искать контакт по имени;
8. если контакт не найден - выводить понятное сообщение;
9. если выбрана неизвестная команда - выводить ошибку и снова показывать меню.
