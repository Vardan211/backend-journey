# HW04: Contact List

## Goal

Build a console contact list where the user can add contacts, list all contacts, and search by name.

## Acceptance Criteria

- [ ] Runs with `dotnet run`
- [ ] Shows the menu until the user chooses `0`
- [ ] Adds a contact with name and phone
- [ ] Handles empty name
- [ ] Handles empty phone
- [ ] Lists all contacts
- [ ] Shows a clear message when the contact list is empty
- [ ] Searches contact by name
- [ ] Shows a clear message when the contact is not found
- [ ] Handles unknown menu command
- [ ] Has at least 3 small commits
- [ ] PR includes reflection answers

## Reflection Questions

1. Why does this program need `List<string>`?
2. Why is a loop useful for the menu?
3. What happens when the user chooses `0`?
4. How did you test contact search?
5. Which part of the assignment was hardest?
