# HW03: Number Guessing Game

## Цель

Сделать консольную игру "Угадай число", где программа загадывает число, а пользователь пытается его угадать.

Это задание тренирует:

- `Random`;
- `while`;
- счетчик попыток;
- сравнение чисел;
- повторение ввода до правильного ответа.

## Теория Перед Практикой

Перед кодом нужно понять:

1. `Random` нужен, чтобы программа могла выбрать случайное число.
2. `while` повторяет код, пока условие истинно.
3. Счетчик попыток - это обычная переменная, которую увеличивают после каждой попытки.
4. Если пользователь ввел не число, это не должно считаться успешной попыткой.

Мини-пример для понимания, не готовое решение:

```csharp
var count = 0;

while (count < 3)
{
    count++;
    Console.WriteLine(count);
}
```

## Что Нужно Сделать

Программа должна:

1. загадать случайное число от `1` до `100`;
2. попросить пользователя угадать число;
3. если пользователь ввел не число - вывести ошибку и спросить снова;
4. если число меньше загаданного - вывести `Больше`;
5. если число больше загаданного - вывести `Меньше`;
6. если число угадано - вывести поздравление и количество попыток;
7. после победы завершить программу.

Пример запуска:

```text
Я загадал число от 1 до 100.
Твоя попытка:
50
Больше
Твоя попытка:
75
Меньше
Твоя попытка:
60
Правильно! Количество попыток: 3
```
