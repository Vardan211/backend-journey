# HW03: Number Guessing Game

## Goal

Build a console game where the program picks a random number and the user keeps guessing until the answer is correct.

## Acceptance Criteria

- [ ] Runs with `dotnet run`
- [ ] Picks a random number from `1` to `100`
- [ ] Lets the user enter multiple guesses
- [ ] Handles invalid input like `abc`
- [ ] Prints `Больше` when the guess is too low
- [ ] Prints `Меньше` when the guess is too high
- [ ] Prints the number of attempts when the user wins
- [ ] Stops after the correct answer
- [ ] Has at least 3 small commits
- [ ] PR includes reflection answers

## Reflection Questions

1. What does the `while` loop do in your program?
2. When does the attempt counter increase?
3. What happens if the user enters `abc`?
4. How does the program know when to stop?
5. Which line of your code are you least sure about?
