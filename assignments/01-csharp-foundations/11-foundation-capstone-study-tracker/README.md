# HW11: Foundation Capstone - Study Tracker Console

## Цель

Сделать итоговое консольное приложение модуля 01: трекер учебных задач, где нужно использовать почти все изученное в основах C#.

Это задание проверяет:

- ввод пользователя;
- `if` / `else`;
- `switch`;
- циклы;
- `List<T>`;
- `Dictionary<TKey, TValue>`;
- методы;
- классы и объекты;
- чтение и запись файла;
- простую валидацию.

## Теория Перед Практикой

Перед кодом нужно повторить:

1. `List<StudyTask>` хранит много задач.
2. `StudyTask` - это класс для одной учебной задачи.
3. `Dictionary<string, int>` удобно использовать для статистики по предметам.
4. Файл нужен, чтобы задачи не пропадали после завершения программы.
5. Методы нужны, чтобы не писать всю программу одним большим куском.

## Что Нужно Сделать

Программа должна показывать меню:

```text
1. Добавить задачу
2. Показать все задачи
3. Показать задачи по предмету
4. Отметить задачу выполненной
5. Удалить задачу
6. Показать статистику по предметам
7. Сохранить задачи в файл
8. Загрузить задачи из файла
0. Выйти
```

Нужно создать класс `StudyTask` со свойствами:

- `Title`;
- `Subject`;
- `IsDone`.

Программа должна:

1. повторять меню, пока пользователь не выберет `0`;
2. хранить задачи в `List<StudyTask>`;
3. добавлять задачу с названием и предметом;
4. не добавлять задачу с пустым названием;
5. не добавлять задачу с пустым предметом;
6. показывать все задачи;
7. показывать задачи только по выбранному предмету;
8. отмечать задачу выполненной;
9. удалять задачу;
10. показывать статистику: сколько задач по каждому предмету;
11. сохранять задачи в файл `study-tasks.txt`;
12. загружать задачи из файла `study-tasks.txt`;
13. не падать, если файла еще нет;
14. обрабатывать неизвестную команду как ошибку;
15. вынести минимум 6 действий в отдельные методы.

Формат файла можно сделать простым:

```text
Math|Read chapter 1|false
CSharp|Finish HW10|true
```

## Как Создать Проект

Из корня репозитория:

```powershell
cd assignments/01-csharp-foundations/11-foundation-capstone-study-tracker
dotnet new console --output .
dotnet run
```

Не делать API, базу данных, EF Core и Docker. Это финальная проверка именно консольного C# foundation. После нее можно переходить к модулю 02: инструменты, ошибки и debugging.
