# HW06: Word Counter Dictionary

## Goal

Build a console word counter that uses `Dictionary<string, int>` to count how many times each word appears.

## Acceptance Criteria

- [ ] Runs with `dotnet run`
- [ ] Asks the user to enter text
- [ ] Handles empty text
- [ ] Converts text to lowercase
- [ ] Splits text into words
- [ ] Counts words with `Dictionary<string, int>`
- [ ] Repeated words increase their counter
- [ ] New words start with counter `1`
- [ ] Empty words are not included
- [ ] Prints the result clearly
- [ ] Has at least 3 small commits
- [ ] PR includes reflection answers

## Reflection Questions

1. What is the key in `Dictionary<string, int>` for this task?
2. What is the value in the dictionary?
3. Why do you need `ContainsKey`?
4. What happens if one word appears 3 times?
5. Which was harder: `List` or `Dictionary`?
