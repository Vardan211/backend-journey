# HW06: Word Counter Dictionary

## Цель

Сделать консольную программу, которая считает, сколько раз каждое слово встречается в тексте.

Это задание тренирует:

- `Dictionary<string, int>`;
- ключ и значение;
- проверку `ContainsKey`;
- разбиение строки через `Split`;
- нормализацию текста через `ToLower`;
- перебор словаря.

## Теория Перед Практикой

Перед кодом нужно понять:

1. `Dictionary<TKey, TValue>` хранит пары: ключ и значение.
2. В этой задаче ключ - слово, значение - количество повторений.
3. Если слово уже есть в словаре, счетчик нужно увеличить.
4. Если слова еще нет, его нужно добавить со значением `1`.

Мини-пример для понимания, не готовое решение:

```csharp
var counts = new Dictionary<string, int>();
counts["cat"] = 1;
counts["cat"]++;
```

## Что Нужно Сделать

Программа должна:

1. попросить пользователя ввести строку текста;
2. если строка пустая - вывести ошибку;
3. привести текст к нижнему регистру;
4. разбить текст на слова;
5. посчитать, сколько раз встречается каждое слово;
6. вывести результат в понятном виде;
7. не считать пустые слова.

Пример:

```text
Введите текст:
cat dog cat

Результат:
cat: 2
dog: 1
```
