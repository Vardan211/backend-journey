# Модуль 01: Основы C#

Цель модуля: научиться писать простые консольные программы, запускать их через `dotnet run`, пользоваться Git и объяснять свой код.

## Задания

1. `01-hello-profile` - первая программа с вводом имени и возраста.
2. `02-simple-calculator` - калькулятор с проверкой чисел, операций и деления на ноль.
3. `03-number-guessing-game` - игра с циклом, случайным числом и счетчиком попыток.
4. `04-contact-list` - список контактов в памяти с меню, добавлением и поиском.
