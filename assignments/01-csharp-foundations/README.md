# Модуль 01: Основы C#

Цель модуля: научиться писать простые консольные программы, запускать их через `dotnet run`, пользоваться Git и объяснять свой код.

## Задания

1. `01-hello-profile` - первая программа с вводом имени и возраста.
2. `02-simple-calculator` - калькулятор с проверкой чисел, операций и деления на ноль.
3. `03-number-guessing-game` - игра с циклом, случайным числом и счетчиком попыток.
4. `04-contact-list` - список контактов в памяти с меню, добавлением и поиском.
5. `05-shopping-list-collections` - отдельная тренировка `List<string>`: добавление, поиск, удаление, очистка.
6. `06-word-counter-dictionary` - подсчет слов через `Dictionary<string, int>`.
7. `07-methods-grade-analyzer` - методы, параметры, возвращаемые значения и анализ оценок.
8. `08-contact-book-classes` - книга контактов на классах, объектах и `List<Contact>`.
9. `09-todo-list-classes` - список задач на `TodoItem`, состоянии и фильтрации.
10. `10-file-notes-app` - заметки с сохранением в файл и чтением после перезапуска.
11. `11-foundation-capstone-study-tracker` - итоговый консольный трекер, который собирает вместе весь foundation.
