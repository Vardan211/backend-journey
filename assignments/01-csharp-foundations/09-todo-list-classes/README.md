# HW09: Todo List With Classes

## Цель

Сделать консольный список задач, где каждая задача - объект `TodoItem`.

Это задание тренирует:

- классы и объекты;
- свойства;
- `List<TodoItem>`;
- изменение состояния объекта;
- фильтрацию списка.

## Теория Перед Практикой

Перед кодом нужно понять:

1. Один класс можно использовать для многих объектов.
2. У объекта может меняться состояние, например `IsDone`.
3. `List<TodoItem>` хранит задачи как отдельные объекты.
4. Фильтр - это перебор списка с условием.

Мини-пример для понимания, не готовое решение:

```csharp
class TodoItem
{
    public string Title { get; set; }
    public bool IsDone { get; set; }
}
```

## Что Нужно Сделать

Программа должна показывать меню:

```text
1. Добавить задачу
2. Показать все задачи
3. Показать только невыполненные
4. Отметить задачу выполненной
5. Удалить задачу
0. Выйти
```

Нужно создать класс `TodoItem` со свойствами:

- `Title`;
- `IsDone`.

Программа должна:

1. повторять меню, пока пользователь не выберет `0`;
2. добавлять задачу через создание объекта `TodoItem`;
3. не добавлять задачу с пустым названием;
4. показывать все задачи;
5. показывать только невыполненные задачи;
6. отмечать задачу выполненной;
7. удалять задачу;
8. если задача не найдена - выводить понятное сообщение;
9. неизвестную команду обрабатывать как ошибку.
