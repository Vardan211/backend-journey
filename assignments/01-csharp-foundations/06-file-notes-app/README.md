# HW06: File Notes App

## Цель

Сделать консольное приложение для заметок, которое сохраняет данные в файл и читает их после перезапуска.

Это задание тренирует:

- работу с файлами;
- `File.Exists`;
- `File.ReadAllLines`;
- `File.AppendAllText`;
- меню;
- методы для маленьких действий.

## Теория Перед Практикой

Перед кодом нужно понять:

1. Обычный список в памяти пропадает после завершения программы.
2. Файл позволяет сохранить данные между запусками.
3. Перед чтением файла нужно проверить, существует ли он.
4. Логику меню удобнее разбить на маленькие методы.

Мини-пример для понимания, не готовое решение:

```csharp
var path = "notes.txt";

if (File.Exists(path))
{
    var lines = File.ReadAllLines(path);
    Console.WriteLine(lines.Length);
}
```

## Что Нужно Сделать

Программа должна показывать меню:

```text
1. Добавить заметку
2. Показать все заметки
3. Найти заметку по слову
0. Выйти
```

Программа должна:

1. повторять меню, пока пользователь не выберет `0`;
2. добавлять заметку в файл `notes.txt`;
3. не добавлять пустую заметку;
4. показывать все заметки из файла;
5. если файла еще нет - писать, что заметок пока нет;
6. искать заметки по слову;
7. если ничего не найдено - выводить понятное сообщение;
8. неизвестную команду обрабатывать как ошибку;
9. вынести минимум 2 действия в отдельные методы.

