# HW02: Simple Calculator

## Цель

Сделать консольный калькулятор, который читает два числа и операцию, проверяет ввод и выводит результат.

Это задание тренирует:

- типы `string`, `double`, `bool`;
- `if` / `else`;
- `switch`;
- `double.TryParse`;
- проверку ошибок до вычисления.

## Теория Перед Практикой

Перед кодом нужно понять:

1. `Console.ReadLine()` всегда возвращает текст, даже если пользователь ввел число.
2. Чтобы превратить текст в число, нужен `double.TryParse`.
3. `if` нужен, когда программа должна выбрать один путь из нескольких.
4. Делить на ноль нельзя, это отдельная ошибка.

Мини-пример для понимания, не готовое решение:

```csharp
var input = Console.ReadLine();

if (double.TryParse(input, out var number))
{
    Console.WriteLine("Это число");
}
else
{
    Console.WriteLine("Это не число");
}
```

## Что Нужно Сделать

Программа должна:

1. спросить первое число;
2. спросить операцию: `+`, `-`, `*` или `/`;
3. спросить второе число;
4. если первое число введено неверно - вывести ошибку;
5. если второе число введено неверно - вывести ошибку;
6. если операция не одна из `+`, `-`, `*`, `/` - вывести ошибку;
7. если пользователь делит на `0` - вывести ошибку;
8. иначе вывести результат.

Пример нормального запуска:

```text
Введите первое число:
10
Введите операцию (+, -, *, /):
/
Введите второе число:
2
Результат: 5
```

Пример ошибки:

```text
Введите первое число:
10
Введите операцию (+, -, *, /):
/
Введите второе число:
0
Ошибка: делить на ноль нельзя.
```
