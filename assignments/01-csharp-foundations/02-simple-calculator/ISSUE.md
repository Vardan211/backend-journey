# HW02: Simple Calculator

## Goal

Build a console calculator that reads two numbers and an operation, validates input, and prints the result.

## Acceptance Criteria

- [ ] Runs with `dotnet run`
- [ ] Asks for the first number
- [ ] Asks for the operation: `+`, `-`, `*`, `/`
- [ ] Asks for the second number
- [ ] Handles invalid first number like `abc`
- [ ] Handles invalid second number like `abc`
- [ ] Handles unknown operation like `%`
- [ ] Handles division by `0`
- [ ] Calculates addition correctly
- [ ] Calculates subtraction correctly
- [ ] Calculates multiplication correctly
- [ ] Calculates division correctly
- [ ] Has at least 3 small commits
- [ ] PR includes reflection answers

## Reflection Questions

1. What is the difference between `string` and `double`?
2. Why is `TryParse` safer than `Parse` here?
3. What happens if the user enters `%` as the operation?
4. How did you test division by zero?
5. Which line of your code are you least sure about?
